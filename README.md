## 社区
加入jdbc依赖，程序会自动使用 HikariCP 作为默认连接池
```yml
<dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-jdbc</artifactId>
</dependency>
或
<dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-jpa</artifactId>
</dependency>
```
